# Pochi.po Frontend

Pochi.po is an AI-powered agent that scans Twitter for emerging meme trends, evaluates their viral potential, and instantly mints a token on Ethereum. It auto-posts the meme token, shares insights, and lets users auto-buy via MPC wallets. Meme. Mint. Moon. üöÄ #AI #Crypto #MemeCoins
